Retrouver ou créer une clé YouTube API

Cette clé sert uniquement à activer la recherche YouTube intégrée. Tu peux aussi continuer à coller des liens YouTube sans clé API.

Pour retrouver une clé déjà créée

  1. Ouvre la page officielle Identifiants / Credentials de Google Cloud.
  2. Vérifie que tu es connecté au bon compte Google.
  3. En haut de Google Cloud, sélectionne le bon projet.
  4. Dans la liste des clés, cherche une clé nommée par exemple YouTube, Blindtest ou API key.
  5. Copie la clé, puis colle-la dans le champ de l’application.
Ouvrir les identifiants Google Cloud

Pour créer une nouvelle clé

  1. Ouvre Google Cloud et crée ou sélectionne un projet.
  2. Active l’API YouTube Data API v3 pour ce projet.
  3. Va dans APIs et services → Identifiants.
  4. Clique sur Créer des identifiants → Clé API.
  5. Copie la clé générée.
  6. Dans l’idéal, restreins la clé à ton domaine de site pour éviter qu’elle soit utilisée ailleurs.
Activer YouTube Data API v3 Créer une clé API

Conseil important

Ne publie jamais ta clé API dans un message, un document ou un dépôt public. Pour un site web, utilise une clé restreinte par domaine HTTP.

Éviter les pubs pendant un blindtest

Pour limiter les interruptions publicitaires pendant la soirée, utilise le navigateur recommandé ci-dessous sur l’appareil de l’arbitre.

Solution recommandée

Avant le blindtest, installe Free Adblocker Browser sur l’appareil utilisé par l’arbitre, puis ouvre l’application Blindtest depuis ce navigateur.

Installer Free Adblocker Browser

Comment l’utiliser

  1. Installe Free Adblocker Browser.
  2. Ouvre l’application Blindtest dans ce navigateur.
  3. Recharge la page admin.
  4. Lance quelques morceaux préparés avant l’arrivée des joueurs pour vérifier que les vidéos démarrent correctement.

Important

Une clé API YouTube sert seulement à rechercher les vidéos dans l’application. Elle ne supprime pas les publicités. Le navigateur anti-pub doit être utilisé directement par l’arbitre pour que cela puisse aider pendant la lecture.
